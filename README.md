# Chào mừng đến với Website của tôi! 🚀

Đây là dự án website cá nhân được lưu trữ trên **GitHub Pages**.

### 🌟 Tính năng chính:
- Giao diện phản hồi (Responsive Design) tương thích mọi thiết bị.
- Nội dung giới thiệu bản thân và các dự án đang thực hiện.
- Tối ưu hóa tốc độ tải trang và trải nghiệm người dùng.

### 🛠 Công nghệ sử dụng:
- HTML5 & CSS3
- JavaScript (ES6+)
- GitHub Pages Deployment

### 📂 Cách chạy dự án:
1. Clone repository này về máy.
2. Mở file `index.html` trên trình duyệt bất kỳ.
